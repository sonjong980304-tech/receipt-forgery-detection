# receipt-forgery-detection
최근 AI 이미지 생성 기술의 발전으로 영수증 위조가 점점 정교해지고 있다. 기업 경비 처리, 보험금 청구, 세금 공제 등 다양한 분야에서 위조 영수증이 악용되며, 회계 법인과 감사 기관에서는 이를 자동으로 탐지할 수 있는 기술적 해결책을 필요로 하고 있다. 본 연구는 FinDit2 데이터셋을 활용하여 이미지 딥러닝 기반의 위조 영수증 탐지 시스템을 설계하고 구현하였다. 비지도학습(오토인코더) 방식과 지도학습(EfficientNet-B3) 방식을 비교 실험하여 각 접근법의 한계와 가능성을 분석하였으며, 다양한 전처리 채널 조합에 대한 Ablation Study를 통해 최적의 전처리 구성을 탐색하였다.
